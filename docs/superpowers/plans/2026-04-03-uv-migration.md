# uv Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate from pip/setuptools to uv with lockfile-based supply chain protection (cryptographic hashes on all direct + transitive deps).

**Architecture:** Replace setuptools build backend with `uv_build`, add `uv.lock` with platform-restricted resolution (Linux + macOS, x86_64 + arm64), update CI workflows and Dockerfile to use `uv sync --frozen` for hash-verified installs. Keep `[project.optional-dependencies]` for pip backward compat.

**Tech Stack:** uv 0.7.x, `uv_build` backend, GitHub Actions `astral-sh/setup-uv@v5`

**Spec:** `docs/superpowers/specs/2026-04-03-uv-migration-design.md`

**Worktree:** `../feat-viraatc-uv` (branch `feat/viraatc-uv`)

### Task 5: Update Dockerfile.dev

**Files:**

- Modify: `scripts/Dockerfile.dev`

- [ ] **Step 1: Rewrite Dockerfile.dev**

Replace the entire file with:

```dockerfile
# Development Dockerfile for the MLPerf Inference Endpoint Benchmarking System
# From project root:
# docker build -f scripts/Dockerfile.dev --build-arg USER_ID=$(id -u) --build-arg GROUP_ID=$(id -g) -t inference-endpoint-dev .
# docker run -v $(pwd):/mnt/inference-endpoint -it --shm-size=512m inference-endpoint-dev bash

FROM python:3.12.11-slim

# Copy uv binary from official image
COPY --from=ghcr.io/astral-sh/uv:0.7.6 /uv /uvx /bin/

# Set environment variables
ENV PYTHONUNBUFFERED=1 \
    PYTHONDONTWRITEBYTECODE=1 \
    UV_COMPILE_BYTECODE=1 \
    UV_LINK_MODE=copy

# Install system dependencies
RUN apt-get update && \
    apt-get install -y --no-install-recommends build-essential procps \
    && rm -rf /var/lib/apt/lists/*

RUN mkdir /mnt/inference-endpoint
WORKDIR /mnt/inference-endpoint

# Copy lockfile + project metadata first for Docker layer caching
COPY pyproject.toml uv.lock .python-version ./
COPY src/ ./src/

# Create a non-root user for security
# Accept UID and GID as build arguments to match host user
ARG USER_ID=1000
ARG GROUP_ID=1000
RUN if ! getent group ${GROUP_ID}; then \
        groupadd -g ${GROUP_ID} appuser; \
    fi && \
    useradd -u ${USER_ID} -g ${GROUP_ID} --create-home --shell /bin/bash appuser --no-log-init && \
    chown -R ${USER_ID}:${GROUP_ID} /mnt/inference-endpoint
USER appuser
ENV PATH="/home/appuser/.local/bin:$PATH"

RUN uv sync --frozen --extra dev --extra test
```

Key changes:

- `COPY --from=ghcr.io/astral-sh/uv:0.7.6` brings in uv binary without pip
- `UV_COMPILE_BYTECODE=1` pre-compiles for faster startup
- `UV_LINK_MODE=copy` ensures files are copied not linked (container-safe)
- `COPY pyproject.toml uv.lock .python-version` before `src/` for layer caching
- `uv sync --frozen` enforces lockfile hashes
- Removed `PIP_NO_CACHE_DIR` and `PIP_DISABLE_PIP_VERSION_CHECK` (no pip)
