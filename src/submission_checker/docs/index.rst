Submission Checker
==================

CLI tool for validating MLPerf benchmark submissions.

.. toctree::
   :maxdepth: 2
   :caption: Contents:

   api

Quickstart
----------

Install::

   uv pip install -e ".[dev,docs]"

Run against a submission directory::

   submission-checker check /path/to/submission
